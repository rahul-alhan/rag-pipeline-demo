# Reviewer Handbook

## Triage Workflow

Reviewers receive items in order of priority:
1. P0 — copyright score above 0.85 (24h SLA)
2. P1 — confidence between 0.70 and 0.90 (72h SLA)
3. P2 — appeals from creators (5 business days)

## Decision Categories

- **Approve**: content meets all guidelines.
- **Reject**: clear policy violation; reviewer must select a violation code.
- **Request Edit**: borderline case; creator is given specific guidance.
- **Escalate**: ambiguous case; routed to senior reviewer with notes.

## IP Scanning

The IP scanning subsystem checks submitted assets against:
- registered trademarks database
- known copyrighted media fingerprints
- internal blacklist of repeat offenders

A positive hit on the trademarks database is an automatic Reject.
